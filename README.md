# My 2D Pixel Platformer

A Godot 4 project for a 2D pixel platformer game.

## Project Structure

- `scenes/` → all game scenes
- `scripts/` → GDScript scripts
- `assets/` → sprites, tiles, audio
- `project.godot` → Godot project file

## How to Run

1. Open the project in Godot 4
2. Press Play ▶️ on the main scene
